Assembleur DXF -> DWG - Version Portable
================================================

VERSION: 1.3
DATE: 04/01/2026

LANCEMENT RAPIDE
================================================
Double-cliquez sur Assembleur_DXF_DWG.exe pour lancer l'application

CONTENU
================================================
- Assembleur_DXF_DWG.exe : Application principale
- diagnostic_zoom.py : Diagnostic du problème de zoom
- nettoyer_dxf.py : Nettoyeur de fichiers DXF
- analyse_dxf.py : Analyseur de DXF
- debug_dwg_conversion.py : Diagnostic conversion DWG
- docs/ : Documentation complète

PREREQUIS
================================================
- Windows 7 ou plus recent
- (Optionnel) ODA File Converter pour conversion DWG
- (Optionnel) AutoCAD pour ouverture automatique

UTILISATION
================================================
1. Double-cliquez sur Assembleur_DXF_DWG.exe
2. Selectionnez le dossier source avec les archives
3. Choisissez la destination
4. Cliquez "Lancer"

DEPANNAGE
================================================
CONSULTEZ LES GUIDES:
- docs/DERNIER_RECOURS.md : Solution rapide pour tous les problemes
- docs/DIAGNOSTIC_ZOOM.md : Resolution du probleme de zoom
- docs/PROCEDURE_DIAGNOSTIC_FINAL.md : Procedure complete de diagnostic
- docs/SOLUTION_ZOOM_FINAL.md : Guide technique complet

SCRIPTS PYTHON INCLUS:
- diagnostic_zoom.py : Analyse complete d'un fichier DXF
- nettoyer_dxf.py : Nettoyeur de fichiers DXF
- analyse_dxf.py : Analyseur de fichiers DXF
- debug_dwg_conversion.py : Diagnostic conversion DWG

INSTALLATION
================================================
- Aucune installation requise !
- Copiez le dossier complet sur n'importe quel PC Windows
